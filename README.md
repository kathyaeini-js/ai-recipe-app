# AI Recipe App

A full-stack AI-powered recipe recommendation application.

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- Python (ML model)

## Features
- Ingredient-based recipe suggestions
- Nutrition information
- Favorites and diet planning

## Project Structure
frontend/   - React app  
backend/    - Node + Express API  
ml-model/   - Python ML logic
